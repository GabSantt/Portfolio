# Portfólio Pessoal

Bem-vindo(a) ao meu portfólio!  
Este projeto foi desenvolvido para apresentar minha trajetória acadêmica, habilidades e principais projetos.

---

## Sobre o Projeto

O portfólio tem como objetivo reunir informações sobre minha formação, cursos, certificações e projetos desenvolvidos ao longo da minha jornada.  
A proposta é criar uma página clara, organizada e responsiva, destacando meus trabalhos e objetivos profissionais.

---

## Tecnologias Utilizadas

- **HTML5** — estrutura da página  
- **CSS3** — estilização e responsividade
