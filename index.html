Planeja SJC

Função: Dev Team - Frontend

Problema: Criar uma aplicação para visualização e análise de dados do Censo 2022, com interface moderna e intuitiva.

Abordagem: Desenvolvimento da interface com HTML, CSS e JavaScript, utilizando Flask para integração backend e consumo de APIs.

Tecnologias Utilizadas:HTML5, CSS3, JavaScript, Bootstrap,Tailwind, Flask, GitHub

Resultados: Aplicação funcional com gráficos dinâmicos e visual responsivo, facilitando a análise de dados

Desafios Enfrentados: Integração entre frontend e backend, responsividade e estruturação do layout de forma clara.

Soluções Adotadas: Uso de componentes modulares e testes iterativos de interface para otimizar a experiência do usuário.

Site sobre Cosplay

Função: Desenvolvedora Frontend

Problema: Criar um site informativo sobre o universo cosplay, reunindo conteúdos sobre personagens, dicas de produção, eventos e galeria de referências de forma organizada e acessível.

Abordagem: Desenvolvimento de uma interface leve e visualmente atrativa, utilizando HTML, CSS e Bootstrap.Foram aplicados elementos visuais temáticos, seções bem estruturadas e imagens otimizadas para criar uma experiência imersiva.

Tecnologias Utilizadas: HTML5, CSS3, Bootstrap,Tailwind, GitHub.

Resultados: Site responsivo e fácil de navegar, com conteúdo segmentado e layout atrativo que facilita o aprendizado e a exploração do mundo cosplay.

Desafios Enfrentados: Organização do grande volume de informações e escolha de um estilo visual que representasse bem o tema sem comprometer a legibilidade.

Soluções Adotadas: Criação de seções modulares, otimização das imagens, uso de cores temáticas equilibradas e testes de usabilidade para garantir clareza e fluidez.

Site Empresarial de Impressão 3D

Função: Desenvolvedora Frontend

Problema: Criar um site institucional para uma empresa de impressão 3D, apresentando serviços, portfólio de peças, informações técnicas e canais de contato.

Abordagem: Estruturar o site com design profissional e moderno, empregando HTML, CSS.Foram criadas seções para serviços, galeria de modelos impressos, valores e formulário de orçamento.

Tecnologias Utilizadas: HTML5, CSS3, GitHub.

Resultados: Página clara, objetiva e responsiva, transmitindo profissionalismo e facilitando a conversão de clientes através de um fluxo intuitivo de navegação.

Desafios Enfrentados: Exibir informações técnicas de forma simples e acessível para diferentes públicos.

Soluções Adotadas: Uso de ícones explicativos, organização das seções em blocos visuais, e padronização das cores e tipografia para reforçar a identidade da marca.
